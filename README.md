# Garage Sale Helper

A one-day-use internal app for managing a family garage sale. Snap a photo of an item, get a suggested price from Claude, and log the actual sale outcome.

## Architecture

```
[Android App] ---HTTPS---> [DigitalOcean Droplet] ---> [Anthropic API]
   (2-3 phones)              (FastAPI + SQLite)         (claude-opus-4-7)
                                    |
                                    v
                              [Image cache + SQLite log]
```

## Components

- **backend/** — FastAPI service hosted on DigitalOcean. Receives photos, calls Claude API for identification + pricing, caches results, logs sales.
- **android/** — Native Android app (Kotlin + Jetpack Compose). Camera capture, price display, sale-outcome logging.
- **docs/** — Specifications.

## Project Setup (WSL Ubuntu)

### Prerequisites

```bash
# Verify WSL Ubuntu
lsb_release -a

# Install system tools (one-time)
sudo apt update && sudo apt install -y python3.12 python3.12-venv python3-pip git curl

# Verify Node.js (Claude Code requires Node 18+)
node --version  # if missing: curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash - && sudo apt install -y nodejs

# Install Claude Code
npm install -g @anthropic-ai/claude-code
```

### Clone & initialize

```bash
mkdir -p ~/projects && cd ~/projects
# (copy this project folder here, or git init it)
cd garage-sale-app

# Backend env
cd backend
python3.12 -m venv .venv
source .venv/bin/activate
pip install -U pip
# (requirements.txt to be generated by Claude Code per spec)
cd ..

# Android: open in Android Studio (Windows-side is fine; Android Studio
# in WSL is unusual — recommended to keep android/ checked out under
# Windows filesystem and edit there, or use a shared Git repo).
```

### Running Claude Code on this project

```bash
cd ~/projects/garage-sale-app
claude
```

Then point Claude Code at the relevant spec:

- For backend work: `read docs/backend-spec.md and implement backend/`
- For Android work: `read docs/android-spec.md and implement android/`

### Environment variables

Create `backend/.env` (never commit):

```
ANTHROPIC_API_KEY=sk-ant-...
SHARED_SECRET=<random 32-char string for HMAC>
DATABASE_PATH=./garage_sale.db
CACHE_DIR=./image_cache
PORT=8443
```

Generate `SHARED_SECRET`:

```bash
openssl rand -hex 32
```

## Decommissioning

After the sale:

```bash
# On the droplet
sudo systemctl stop garage-sale
# Optional: snapshot the SQLite DB for records
scp droplet:~/garage-sale-app/backend/garage_sale.db ./sale-results.db
# Destroy droplet via DigitalOcean console
```
